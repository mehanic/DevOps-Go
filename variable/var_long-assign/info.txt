переменные/variables

оперативная память(RAM)
переменная
    тип  -> цифра(1,2,3,4,5,6), строка("Hello","123213"), массив, объект, структура
    значение -> 1,2,3,4, "Привет как дела?"
    адресс -> 0x1232342

цифра,числа -> int, int32, int64
дробные числа, числа с плавающей запятой -> float32, float64
строки -> string
var    a      int
var -> модификатор который показывает что это переменная
a -> название переменной
int -> тип данных

a:=3
a-> переменная
3-> первоначальное значение переменной


invalid operation: a / b (mismatched types float64 and int)
у вас не совместимые для общей операции типы данных

reflect.TypeOf(переменная) -> для того чтобы узнать тип переменной

fmt.Scan(&переменная) -> для того чтобы принять данные из консоли и поместить в переменную
