### Функция `counter`

```go
func counter(x []int, n int) int {
    var count int

    for _, val := range x {
        if n == val {
            count++
        }
    }
    return count
}
```

1. **Параметры функции**:
   - `x []int`: это срез чисел типа `int`. Мы будем искать, сколько раз число `n` встречается в этом срезе.
   - `n int`: это число, которое мы ищем в срезе `x`.

2. **Переменная `count`**:
   - `var count int`: здесь мы создаем переменную `count`, которая будет отслеживать количество вхождений числа `n` в срез `x`. Изначально она равна 0.

3. **Цикл по срезу `x`**:
   - `for _, val := range x`: здесь мы используем цикл `for` с конструкцией `range` для итерации по всем элементам среза `x`. В каждой итерации:
     - `_` — это игнорируемый индекс элемента (мы не используем индекс, нам важны только значения).
     - `val` — это текущий элемент из среза `x`.

4. **Условие для подсчета**:
   - `if n == val`: если текущий элемент `val` в срезе `x` равен числу `n`, то увеличиваем переменную `count` на 1. Это означает, что мы нашли одно вхождение числа `n`.

5. **Возврат результата**:
   - `return count`: после того, как цикл завершит итерацию по всем элементам среза, функция возвращает количество вхождений числа `n` в срез.

### Функция `main`

```go
func main() {
    var x_slice = []int{23, 24, 3, 25, 34, 35, 2, 56, 66, 3, 76, 68, 4, 3, 33, 24, 5, 2}

    fmt.Println("Count :=", counter(x_slice, 3))
}
```

1. **Объявление среза**:
   - `var x_slice = []int{23, 24, 3, 25, 34, 35, 2, 56, 66, 3, 76, 68, 4, 3, 33, 24, 5, 2}` — создаем срез `x_slice` из целых чисел, который содержит 18 чисел.

2. **Вызов функции `counter`**:
   - `fmt.Println("Count :=", counter(x_slice, 3))`: мы вызываем функцию `counter`, передавая ей срез `x_slice` и число `3`. Эта функция вернет количество вхождений числа 3 в срез `x_slice`, и мы выведем это значение на экран.

### Как работает программа:

1. Программа создает срез с 18 элементами.
2. Затем функция `counter` перебирает все элементы среза, и каждый раз, когда встречает число 3, увеличивает счетчик `count`.
3. В данном случае число 3 встречается 3 раза в срезе (на позициях 2, 9 и 13).
4. Функция возвращает `3` как результат.
5. В `main` мы выводим строку `"Count := 3"`, потому что число 3 встречается в срезе 3 раза.

### Пример вывода:

```
Count := 3
```

### Ключевые моменты:
- **Использование среза**: `x []int` — функция работает с срезом, который позволяет передавать переменное количество элементов.
- **Цикл `for range`**: используется для итерации по всем элементам среза.
- **Подсчет вхождений**: при каждом совпадении с искомым значением увеличиваем счетчик `count`.
- **Возврат значения**: функция возвращает количество вхождений числа в срез.
